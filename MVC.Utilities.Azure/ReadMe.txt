You must do the folloing things before this package will work:

  1) Setup your cache on http://portal.appfabriclabs.com/

  2) Open the app.config or web.config

  3) Search for the two instances of "YOURCACHENAME" and replace with the cache name you defined in the portal.

  4) Search for the two instances of "YOURCACHETOKEN" and replace with the cache token acquired in the portal.
